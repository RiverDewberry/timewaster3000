"timewaster3000" was a very simplistic 'game' designed to be a short distraction for when I need small break. Now, I realize that the real time waster is having a fun project that I can work on whenever I'm bored.

The reason for some of the more "interesting" decisions made in the creation of this game
(such as this thing being written in plain JS)
were made such that this can run locally on a school managed Chromebook >:3

### How to run timewaster3000 on a Chromebook:
(note: all you have to do is open the html file, this section is for people who have limited knowledge about computers)

1. click "index.html"
2. click the button to "Wownload raw file"
3. press the shortcut "ctrl+o"
4. find the file that you downloaded and double click on it
5. bookmark the tab that the game is running on so that you can easily access this later
