"timewaster3000" is a very simplistic 'game' designed to be a short distraction for when I need small break.
Hopefully someone else can find use from this as well.

The reason for some of the more "interesting" decisions made in the creation of this game
(such as this thing being written in plain JS)
were made such that this can run locally on a school managed Chromebook >:3

### How to run timewaster3000 on a Chromebook:
(note: all you have to do is open the html file, this section is for people who have limited knowledge about computers)

1. click the green button in the upper left hand corner of the screen labeled "Code"
2. click "Download ZIP"
3. press the shortcut "ctrl+o"
4. find the file that you downloaded and double click on it, then click "Extract all"
5. click twice on the "timewaster3000-main" folder
6. click twice on index.html
7. close the file menu and you should see the game open and running
8. bookmark the tab that the game is running on so that you can easily access this later
